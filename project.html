2.Research Project

This research project explores the impact of heavy traffic on network signal loss in healthcare settings, utilizing a Python programming approach with data analysis and data visualization techniques. Conducted from June 18 to July 7, 2025, working together with Vidhi Dixit, this study aims to identify patterns and correlations between heavy traffic and signal loss, which can compromise patient care and data transmission. By analyzing network traffic data and signal strength metrics, the project provides valuable insights into the critical areas where network optimization can improve signal reliability and overall healthcare service delivery. Through data visualization, the project presents insightful representations of the findings, informing strategies for enhancing network infrastructure and ensuring seamless communication in healthcare environments.
